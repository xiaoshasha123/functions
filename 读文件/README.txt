说明：

1.功能---实现对各种编码格式的txt文件、.doc文件以及.docx文件的读取。

2.材料---readfile.py(实现方法)，antiword(windows下使用)，antiword-0.37.tar.gz(linux下使用)，

         requirements.txt(导入模块)。

3.测试---windows：将readfile.py和antiword(windows)文件夹放在同一目录下,安装所需的模块后可测试。

         linux：需编译安装antiword-0.37.tar.gz，注意路径和权限，修改readfile.py中antiword的路径。
